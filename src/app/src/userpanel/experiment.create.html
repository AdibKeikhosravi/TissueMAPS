Create experiment

To create an experiment, please choose what type of microscope you used to acquire your images.

A well plate represents a container with multiple reservoirs for different samples (wells) that might be stained independently, but imaged under the same conditions. There are different plate *formats*, which encode the number of wells in the plate, e.g. "384".

Please specify the microscope you used to take the pictures for this experiment. Note that all plateSources of an experiment have to be acquired by the same microscope.
